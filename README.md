# Morphous

Morphous is a static TanStack Start catalog of nature and animal inspired design systems. The current repo state ships one regenerated snow-leopard system with a background-free animal cutout, a design-system board, 12 slide preview images, an editable PowerPoint template, and shadcn/tweakcn-compatible theme exports.

## Commands

```bash
pnpm dev
pnpm validate:catalog
pnpm verify:pptx
pnpm typecheck
pnpm lint
pnpm build
```

## Generation

The repo includes a Codex skill at `.codex/skills/morphous-catalog`.

```bash
pnpm generate:catalog
```

Assembles the current snow-leopard bundle from generated image assets that already exist in `public/systems/morphous-snow-leopard`. It derives the shadcn/tweakcn palette from `animal.png`, writes the theme exports and catalog metadata, and rebuilds the editable PowerPoint template.

Image assets are not generated by repo scripts. Generate or edit these images first with the Codex image tool, then copy them into the expected paths:

- `animal.png`
- `design-system.png`
- `slides/01.png` through `slides/12.png`

The assembler exits with a missing-asset error if any required image is absent.

## PPTX Render Verification

```bash
pnpm verify:pptx
```

Exports `public/systems/morphous-snow-leopard/template.pptx` through LibreOffice/PDF/PNG and verifies that every slide is built from editable PowerPoint text and shape objects without full-slide raster screenshots. Rendered PNG previews and the editability report are written to:

```text
public/systems/morphous-snow-leopard/pptx-rendered/
```

## Asset Contract

Each system writes:

```text
public/systems/{slug}/animal.png
public/systems/{slug}/design-system.png
public/systems/{slug}/slides/01.png through slides/12.png
public/systems/{slug}/template.pptx
public/systems/{slug}/theme.css
public/systems/{slug}/theme.json
```
