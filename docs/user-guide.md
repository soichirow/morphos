# 利用ガイド

## 言語

画面右上の「日本語 / English」で切り替えます。日本語では日本語名を主表示し、判別しやすいよう英名も併記します。設定はブラウザに保存されます。

## 探す

検索欄は日本語名、英名、モチーフ、カテゴリ、説明、プロンプト由来の語を対象にします。大分類、細分類、色、並び順も組み合わせられます。「ランダム表示」は現在の絞り込み結果から、選択中とは別のシステムを選びます。

## 共有URL

選択中のデザインは次の形式で共有できます。

```text
https://morphos.so1ro.com/gallery/?system=morphous-abalone
```

「URLをコピー」やSNS共有が作るURLには `system` だけを含めます。検索語、カテゴリ、色、表示モード、お気に入りは共有されません。LLMにはこのURLをそのまま渡せます。

`/systems/<slug>` を開いた場合も、同じ内容を示す `/gallery/?system=<slug>` へ正規化されます。

## お気に入り

お気に入りは `localStorage` にだけ保存します。ログイン、クラウド同期、端末間同期、バックアップはありません。ブラウザデータを消すとお気に入りも消えます。

## 表示モード

### ふわふわモード

初期値です。611システムのモチーフをローカルの抽象化したらくがき画像で表示します。デザインボードなど、らくがき版がない画像はモザイク表示にします。元画像のダウンロードリンクは出しません。

### モザイクモード

通常画像を強くぼかし、彩度とコントラストを落とし、格子を重ねます。画像自体は原作配信元から読み込むため、通信を完全に止めるモードではありません。元画像のダウンロードリンクは出しません。

### 通常モード

原作の画像をそのまま表示します。高解像度表示と画像ダウンロードが使えます。刺激を避けたい場合は選ばないでください。

選択したモードは `morphous:motif-display-mode` としてブラウザに保存されます。ふわふわ表示の大きなモチーフには、利用者が明示的に元画像を見る操作がある場合があります。

## コピーと生成

CSS、theme.json、prompts.json、個別プロンプト、色コードはクリップボードへコピーできます。PowerPointとWordはブラウザ内で生成してダウンロードします。画像アセットのダウンロードは通常モードだけです。

## SNS共有

X、Facebook、LinkedIn、Bluesky、Reddit、WhatsApp、Telegramに対応します。対応端末ではOSの共有メニューも使えます。共有先へ渡すのは選択中システムの名前、案内文、共有URLです。

## プライバシーと分析

言語、お気に入り、表示モード、パレット調整、分析同意をブラウザに保存します。GA4 Measurement IDが設定されていても、分析を許可するまではGoogle Analyticsを読み込みません。詳細はサイト内の [プライバシー方針](https://morphos.so1ro.com/privacy/) と [サイトガバナンス](site-governance.md) を参照してください。
